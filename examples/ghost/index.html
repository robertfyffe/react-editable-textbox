Ghost Theme

This example illustrates the behaviour of the isGhost prop. It provides an editable textbox with just text styling.


Code

const defaultText = `<p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Suspendisse maximus felis ut mauris mattis tempor. Ut suscipit aliquet est non sagittis. Fusce lobortis sodales nulla, tincidunt ornare massa vehicula id. Vivamus sit amet tellus quis risus ultrices iaculis. Praesent a porttitor nisi. Maecenas vitae aliquam sapien.</p>`;

const App = () => <EditableTextBox defaultText={defaultText} isGhost />
